**Flask API for the backend** <br />


**Create a venv from base directory** <br />
pip install venv <br />
source venv/bin/activate <br />
pip install -r requirements.txt <br />


**Run the flask app** <br />
export FLASK_APP=be/api.py <br />
flask run <br />
